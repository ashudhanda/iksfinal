# IKS (Indian Knowledge System) MCQ Quiz App 🌟

An interactive, feature-rich Multiple Choice Question (MCQ) web application built to help students practice and evaluate their knowledge of the Indian Knowledge System (IKS). Designed with a modern, glassmorphism UI, this app ensures an engaging learning experience.

Created by **Ashu Dhanda**.

## ✨ Features
- **155 Verified Questions**: A comprehensive database of IKS questions covering history, philosophy, astronomy, metallurgy, and more.
- **Dynamic Themes**: Toggle between a beautiful Dark Mode with a video/starry canvas background and a warm Light Mode.
- **Background Controls**: Stop the background video anytime to reveal a subtle, starry night animation.
- **Sound Effects**: Interactive audio feedback for correct/wrong answers (toggleable).
- **Custom Fonts**: Choose between a futuristic 'ZeroArea' font or a readable 'Normal' font.
- **Timer Mode**: Challenge yourself with a countdown timer for each question.
- **Detailed Score Report**: After finishing the quiz, view your performance analytics, total score, and a complete Answer Key to review mistakes.

## 🚀 Technologies Used
- HTML5
- CSS3 (Vanilla, Glassmorphism design, CSS Variables)
- JavaScript (ES6+, DOM Manipulation, Canvas API)

## 📦 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/iks-quiz-app.git
   ```
2. Open `index.html` in any modern web browser.
3. Enjoy the quiz!
